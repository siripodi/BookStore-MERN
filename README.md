# 📚 BookStore MERN Application

## 📖 Project Overview

The BookStore MERN Application is a full-stack web application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). It allows users to browse books, search books, add books to a wishlist, place orders, and track their orders. Sellers can manage books, while administrators can manage users, sellers, books, and orders.
